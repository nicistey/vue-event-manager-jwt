## Технологии

- Vue.js
- Vuex
- Vue Router
- Axios
- JWT

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/nicistey/vue-event-manager-jwt.git
   cd vue-event-manager-jw
   ```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
